Password Strength Checker
This is a web-based tool designed to help users assess the strength of their passwords and check if their passwords have been exposed in known data breaches. It provides instant feedback and suggestions for creating more secure passwords, aiming to improve overall security awareness.

Live Demo
You can try out the live version of the project here: Password Strength Checker

Screenshots
Key Features
Real-time Password Strength Analysis: Evaluates passwords based on multiple criteria:
Minimum length (8+ characters)
Inclusion of lowercase letters
Inclusion of uppercase letters
Inclusion of numbers
Inclusion of special characters
Data Breach Check: Integrates with the 'Have I Been Pwned' API to check if a password has been compromised in any known data breaches.
Constructive Feedback: Offers actionable tips to help users improve their password strength.
Estimated Crack Time: Provides an estimation of how long it would take to crack the given password.
Responsive Design: Ensures a seamless user experience across a wide range of devices, including desktops and mobile phones.
Tech Stack
HTML5: The core markup language for the web pages.
CSS3: Used for styling and creating a responsive layout.
JavaScript: Powers the application's logic, including password validation and API integration.
